# Code of Conduct

This project welcomes professional, respectful collaboration.

Expected behavior:

- Be clear, constructive, and specific.
- Respect different field backgrounds and test environments.
- Discuss technical disagreement with evidence, not personal attacks.
- Keep safety-critical wording conservative and precise.

Unacceptable behavior:

- Harassment, insults, or discriminatory comments.
- Publishing private vulnerability details before maintainers can respond.
- Misrepresenting this lab simulator as a certified timing source.

Maintainers may edit, hide, or remove comments that make the project unsafe or unproductive.
