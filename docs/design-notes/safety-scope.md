# Safety Scope

Process Bus Timing Lab / PTP Lab Clock is a **synthetic lab simulator and diagnostic companion**.

It may be used for:

- Process Bus Analyzer validation.
- Wireshark-visible PTP packet generation.
- Quick PTP visibility check.
- IED response observation in a controlled lab.
- Pdelay exchange simulation.
- Fault scenario testing.
- Training and demo.

It must not be used or marketed as:

- GPS/PTP grandmaster replacement.
- Certified timing reference.
- Nanosecond or microsecond accuracy validation source.
- Official FAT/SAT timing source.
- Protection/metrology acceptance reference.

When in doubt, label output as **lab simulation only**.
