# Limitations

Process Bus Timing Lab is intentionally scoped as a lab simulator and diagnostic companion.

It does not provide:

- hardware timestamping,
- clock servo discipline,
- GPS/IRIG input,
- BMCA-complete grandmaster behavior,
- conformance certification,
- relay-acceptance timing guarantees.

Use it to validate PTP visibility, analyzer decoding, fault symptoms, and evidence collection. Use certified PTP grandmaster equipment for protection, metering, and final commissioning workflows.
